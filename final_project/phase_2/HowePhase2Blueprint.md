## 5. OOP Class Design — Phase 2 Additions

### 5.1 `Admin.add_student()` — Implemented

```python
def add_student(self):
    from HoweUtil import (load_students, is_value_unique,
                          save_row_to_csv, STUDENTS_CSV_PATH, STUDENT_HEADERS)
    while True:
        # collect all 5 fields
        first_name = input("  First name: ").strip()
        last_name  = input("  Last name: ").strip()
        username   = input("  Username: ").strip()
        password   = input("  Password: ").strip()
        email      = input("  Email: ").strip()

        # validate no empty fields
        if not all([first_name, last_name, username, password, email]):
            print("  [!] All fields are required. Please try again.")
            continue

        # enforce unique username — re-prompt username ONLY if duplicate
        students = load_students()
        while not is_value_unique(username, students, 2):
            print(f"  [!] Username '{username}' already exists. Choose a different username.")
            username = input("  Username: ").strip()
            if not username:
                print("  [!] Username cannot be empty.")

        save_row_to_csv(STUDENTS_CSV_PATH, STUDENT_HEADERS,
                        [first_name, last_name, username, password, email])
        print(f"  [✓] Student '{username}' added successfully.")

        again = input("  Add another student? (yes/no): ").strip().lower()
        if again != "yes":
            break
```

### 5.2 `Admin.add_course()` — Implemented

```python
def add_course(self):
    from HoweUtil import (load_courses, is_value_unique,
                          save_row_to_csv, COURSES_CSV_PATH, COURSE_HEADERS)
    while True:
        course_number = input("  Course number: ").strip()
        course_title  = input("  Course title: ").strip()

        if not all([course_number, course_title]):
            print("  [!] All fields are required. Please try again.")
            continue

        # enforce unique course number — re-prompt course_number ONLY if duplicate
        courses = load_courses()
        while not is_value_unique(course_number, courses, 0):
            print(f"  [!] Course number '{course_number}' already exists.")
            course_number = input("  Course number: ").strip()
            if not course_number:
                print("  [!] Course number cannot be empty.")

        save_row_to_csv(COURSES_CSV_PATH, COURSE_HEADERS, [course_number, course_title])
        print(f"  [✓] Course '{course_number}' added successfully.")

        again = input("  Add another course? (yes/no): ").strip().lower()
        if again != "yes":
            break
```

## 6. Algorithm Design

### 6.1 Admin Menu Algorithm (Phase 2 version)

```algorithm
After successful login (admin_obj is authenticated Admin instance):

Repeat:
    Display admin menu:
        1. Add a new student
        2. Add a new course
        3. Exit

    Prompt for choice

    If choice = "1": admin_obj.add_student()
    Else if choice = "2": admin_obj.add_course()
    Else if choice = "3": print "Goodbye." → sys.exit()
    Else: print "Invalid choice."

Until user exits
```

> Note: Menu expands to 6 options in Phase 3.

---

### 6.2 Uniqueness Check (from HoweUtil.py)

```algorithim
Function is_value_unique(value, data, column_index):
    For each row in data:
        If row[column_index].strip().lower() == value.strip().lower():
            Return False   # duplicate found
    Return True
```

**Key design decision:** Duplicate username → re-prompt username only (not all 5 fields). Duplicate course number → re-prompt course number only.

---

### 6.3 Save Row to CSV (from HoweUtil.py)

```algorithim
Function save_row_to_csv(filepath, headers, new_row):
    If file does not exist:
        Open in write mode
        Write headers row
        Write new_row
    Else:
        Open in append mode
        Write new_row only
```

## 8. Edge Cases & Validation

| Scenario | Handling |
| --- | --- |
| Any required field is empty | Print error, re-prompt entire entry (continue loop) |
| Username already exists | Print error, re-prompt username only (inner while loop) |
| Course number already exists | Print error, re-prompt course number only (inner while loop) |
| `students.csv` doesn't exist on first run | `load_students()` returns `[]`; `save_row_to_csv()` creates file with headers |
| `courses.csv` doesn't exist on first run | Same pattern |
| Admin enters "no" to "add another?" | Exit the add loop, return to main menu |
| Invalid menu choice | Print "Invalid choice." and re-display menu |
| Extra whitespace | `.strip()` on all inputs |
