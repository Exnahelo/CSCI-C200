Purpose
Lots of software applications solve particular problems through using formulae, e.g. on an online conversion web site (https://www.onlineconversion.com/), you can perform length conversion, weight conversion etc.  The program will feed input(s) into a certain formula and display a result. This lab will help you understand how data is stored and send into a formula and how to use expression(s) to format a formula. You are actually making a real software application, maybe a small one, but a real one!
Description
In this lab, you will create a Python program that solves a problem using a formula that contains variables and expression(s). Find a problem that you have solved before using other methods, such as on paper, using a calculator, using a web application. Here are a few examples - converting a temperature in degrees Fahrenheit to degrees Celsius, converting a US dollar amount to an amount in another currency, finding the area of a triangle, calculating the force of gravity, finding the future value with compound interest etc.   All the values used in the formula must be stored in variables. The output of the program is a complete sentence that states the input value(s) and the result, e.g. 32 degrees Fahrenheit is 0 degrees Celsius.
Learning Outcomes
Upon successful completion of this lab, you will be able to

	•	Understand the purpose of using variables

	•	Use a formula to form an expression in which variables are used
	•	Properly use operators such as a mathematical operator (+, -, *, /), an assignment operator, a string concatenation operator
	•	Understand the purpose of the str() function and use it where necessary
	•	Use block comments or single line comments to well organize code blocks
	•	Follow rules for good coding practice
