Purpose
In this lab, you will practice using loops to create arrays and retrieving values from arrays. You will learn to store and use real world data in a structured way. 

Description
Get user inputs for different pieces of related information and create arrays to store them in a structured way.  Here are two visual examples what your data look like – 
	•	The highest-grossing American films released in 2025

















	The above data can be stored in two different ways: 1) using several 1-D arrays, such as use 4 arrays (mimicking 4 columns) and each array has 10 elements(mimicking 10 rows), or stored in 10 arrays and each array has 4 elements.  2)using a 2-D array. You decide how you want to store data. 





	•	Indianapolis Colts Player 2025

	Again the above data can be stored in several 1-D arrays or a 2-D array. You decide how to store data.  

All the data will be entered by the user, until the user enters “done” or another indicator. After all the data are stored in arrays, your program will loop through the arrays and output all the information, plus one more piece of information computed from existing data, e.g. in the Colts player example,  besides outputting weight of each player in pounds, you will also output weight in kilograms. 

You don’t have to use the above examples. Whatever information you decide to use, your program must simulate a real-world data collection process, e.g. a human resource application asking user to enter employee information, a program for librarians to enter book information. 

Rubric
