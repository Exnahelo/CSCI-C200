Purpose
Through this lab, you will practice how to use a Python program to extra information from a string with a particular pattern, which is common in software applications that work with a lot of text values. 
Learning Outcome
Upon successful completion of this lab, you will be able to
	•	Understand string index values
	•	Understand string slicing
	•	Create functions that return values
Description
Think of a situation where a section of a string (a sub string) has a particular meaning, or is often extracted to identify something. Below are some examples:
	•	In a full name, the part after space is the last name.
	•	In a phone number such as (317)123-4567,  the three digits after “(“ is an area code. 
	•	In a long version of a zip code, such as 46202-4218, the last part sometimes needs to be removed. 
	•	The last four digits of a social security number are sometimes extracted to identify a person. 
	•	In IU Indianapolis, a room label such as SL155 can be split into a building number and a room number. 
Determine the type of string you want to process, then create a Python program that asks a user to enter a value, split the string, and output a sentence stating original input and the split result. If the string can be split into two parts, and each part means something, the output should include both parts, for instance, a full name can be split into a first name and a last name, the output sentence will be “In the name ‘John Smith’ , ‘John’ is the first name and ‘Smith’ is the last name”. If the task is to remove a part of the string, the output can include only the remaining part, for instance, “The short version of the zip code 46202-4218 is 46202”. 
User will be asked if they want to try again, for unlimited times. Design a way to end the program based on user input, e.g. enter “yes” to continue and enter “no” to end. When you ask the user to enter a string value, be sure to tell user what the format should be, e.g. if asking user to enter a phone number, specify the phone number format. There is no need to validate the format. You can assume the user entered the value in the right format. String processing should be defined as a function that returns a value, e.g. to find the first name of a string, the function can be defined as findFirstName(name) where name is the user entry, and the return value is the first name.
