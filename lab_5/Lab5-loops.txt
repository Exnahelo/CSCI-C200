Description
Design a program that will perform a task based on user inputs. User will be asked to enter value(s) and select from a list of choices, e.g. if your program is about length conversion, the users will be asked to enter a number in centimeter, then be asked to choose what unit of measurement to convert into, and the choice can be: 
	•	meter
	•	millimeter
	•	inch
	•	foot
…

The above are requirements from Lab 4. For this lab, you will add the following features: 
If user has entered an invalid entry (e.g. user did not enter a numeric value as required, either entry is not numerical, or out of the desired range, or not one of the allowed entries), user will be asked to try again, for up to 5 times. After 5 times, program will exit with a message. If all entries are valid, program will output a message with input values and final result, e.g. 20 centimeters is equal to 0.2 meters. 

A sample program (loginControl.py, verify.py) can be found in Canvas-Files-Labs-Lab 5 folder. Download both files, upload them to your program space(either on pythonanywhere.com or your local computer), and run the loginControl.py file to see the behavior.
